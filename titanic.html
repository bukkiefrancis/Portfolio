Titanic Survival Analysis

This project explores survival patterns from the Titanic disaster using SQL, Excel and data visualisation techniques. The aim was to understand how factors such as gender, passenger class, age and family size influenced survival outcomes.

Introduction

The Titanic dataset is widely used in data analysis as it contains multiple variables that may influence survival outcomes. This project investigates patterns within the dataset to better understand how social and demographic factors affected survival during the disaster.

By combining Excel analysis with SQL queries, the dataset was explored systematically to uncover meaningful insights and communicate them clearly.

Objectives

  • Explore survival patterns within the Titanic dataset.
  • Identify which variables influenced survival outcomes.
  • Use SQL and Excel analysis to uncover trends.

Methods Used

  • Data cleaning and preparation in Excel
  • Exploratory analysis using pivot tables
  • SQL queries for grouped statistics
  • Power BI dashboard creation

SQL Analysis

SQL queries were used to validate the dataset, analyse survival patterns and calculate key statistics across different passenger groups.

Survival Rate by Gender

SELECT
  Sex,
  COUNT(*) AS passengers,
  ROUND(AVG(Survived) * 100, 0) AS survival_rate
FROM titanic
GROUP BY Sex;

Women and Children First Analysis

SELECT
  CASE
      WHEN "age group" IN ('Child', 'Teen') THEN 'Children'
      WHEN Sex = 'female' THEN 'Women'
      ELSE 'Adult Men'
  END AS passenger_group,
  COUNT(*) AS passengers,
  ROUND(AVG(Survived) * 100, 0) AS survival_rate
FROM titanic
GROUP BY passenger_group
ORDER BY survival_rate DESC;

Survival by Class, Gender and Age

SELECT
  CASE
      WHEN Pclass = 1 THEN 'First Class'
      WHEN Pclass = 2 THEN 'Second Class'
      ELSE 'Third Class'
  END AS passenger_class,
  Sex,
  "age group",
  COUNT(*) AS passengers,
  ROUND(AVG(Survived) * 100, 0) AS survival_rate
FROM titanic
GROUP BY passenger_class, Sex, "age group"
ORDER BY survival_rate DESC;

Family Size Analysis

SELECT
  CASE
      WHEN "family size" = 1 THEN 'Alone'
      WHEN "family size" BETWEEN 2 AND 4 THEN 'Small Family'
      ELSE 'Large Family'
  END AS family_group,
  COUNT(*) AS passengers,
  ROUND(AVG(Survived) * 100, 0) AS survival_rate
FROM titanic
GROUP BY family_group
ORDER BY survival_rate DESC;

Ranking

SELECT
  Pclass,
  Sex,
  COUNT(*) AS passengers,
  ROUND(AVG(Survived) * 100, 0) AS survival_rate,
  RANK() OVER (ORDER BY AVG(Survived) DESC) AS survival_rank
FROM titanic
GROUP BY Pclass, Sex;

Findings

Key Insights

  • Female passengers had a much higher survival rate than male passengers (~74% vs ~19%).
  • First-class passengers had the highest survival rate (~63%), compared to second (~47%) and third (~24%).
  • Children showed better survival outcomes than adult males.
  • Passengers travelling alone generally had lower survival rates.
  • Survival was influenced by a combination of gender, class and age.

Conclusion

This analysis shows that survival during the Titanic disaster was strongly influenced by social and demographic factors rather than chance alone.
